+------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| O QUE TORNA UM DASHBOARD PROFISSIONAL?                                                                                                                                 |
+------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Estado centralizado: Pinia store é a única fonte da verdade. Widgets não buscam dados individualmente — todos leem do mesmo store, garantindo consistência visual.     |
|                                                                                                                                                                        |
| Dados pré-computados no backend: Nenhum widget faz cálculo de agregação no frontend. GROUP BY, percentuais, streaks — tudo vem pronto do schema analytics.*.           |
|                                                                                                                                                                        |
| Atualização cirúrgica via WebSocket: O evento .metrics.updated substitui apenas o estado do store. Vue reatividade propaga a mudança para todos os widgets sem reload. |
|                                                                                                                                                                        |
| Loading states granulares: Cada widget exibe skeleton loader independente. O dashboard nunca trava esperando todos os dados ao mesmo tempo.                            |
|                                                                                                                                                                        |
| Composables como camada de lógica: Views são dumb. useDashboard() encapsula fetch, WebSocket, estado de erro e lógica de refresh. Reutilizável e testável.             |

2. Analytics Store (Pinia) — Fonte Única da Verdade

O store é o coração do dashboard. Todos os widgets leem exclusivamente dele via computed properties reativas. Quando o WebSocket atualiza o store, Vue propaga automaticamente para todos os consumidores.

+-------------------------------------------------------------------------------------------------------+
| // src/stores/analytics.store.ts                                                                      |
|                                                                                                       |
| import { defineStore } from 'pinia';                                                                  |
|                                                                                                       |
| import { ref, computed } from 'vue';                                                                  |
|                                                                                                       |
| import type { DashboardData, UserMetrics, TechMetrics, TimeSeriesPoint } from '@/types/domain.types'; |
|                                                                                                       |
| import { analyticsApi } from '@/api/modules/analytics.api';                                           |
|                                                                                                       |
| export const useAnalyticsStore = defineStore('analytics', () => {                                     |
|                                                                                                       |
| // ── RAW STATE ────────────────────────────────────────────────────                                  |
|                                                                                                       |
| const dashboard = ref<DashboardData | null>(null);                                                    |
|                                                                                                       |
| const isLoading = ref(false);                                                                         |
|                                                                                                       |
| const isRecalculating= ref(false); // spinner WebSocket                                               |
|                                                                                                       |
| const error = ref<string | null>(null);                                                               |
|                                                                                                       |
| const lastFetchAt = ref<Date | null>(null);                                                           |
|                                                                                                       |
| const selectedPeriod = ref<'7d' | '30d' | '90d'>('30d');                                              |
|                                                                                                       |
| const STALE_MS = 5 * 60 * 1000; // 5 minutos (igual ao TTL do backend)                                |
|                                                                                                       |
| // ── COMPUTED (granular — cada widget consume o que precisa) ──────                                  |
|                                                                                                       |
| const userMetrics = computed(() => dashboard.value?.user_metrics ?? null);                            |
|                                                                                                       |
| const techMetrics = computed(() => dashboard.value?.technology_metrics ?? []);                        |
|                                                                                                       |
| const timeSeries = computed(() => dashboard.value?.time_series_30d ?? []);                            |
|                                                                                                       |
| const topTechs = computed(() => dashboard.value?.top_technologies ?? []);                             |
|                                                                                                       |
| const weeklyComp = computed(() => dashboard.value?.weekly_comparison ?? null);                        |
|                                                                                                       |
| // Streak formatado para o KpiCard                                                                    |
|                                                                                                       |
| const currentStreak = computed(() => ({                                                               |
|                                                                                                       |
| value: userMetrics.value?.current_streak_days ?? 0,                                                   |
|                                                                                                       |
| label: userMetrics.value?.current_streak_days === 1 ? 'dia' : 'dias',                                 |
|                                                                                                       |
| isPersonalBest: (userMetrics.value?.current_streak_days ?? 0) ===                                     |
|                                                                                                       |
| (userMetrics.value?.max_streak_days ?? 0),                                                            |
|                                                                                                       |
| }));                                                                                                  |
|                                                                                                       |
| // Total de horas formatado                                                                           |
|                                                                                                       |
| const totalHoursFormatted = computed(() => {                                                          |
|                                                                                                       |
| const h = userMetrics.value?.total_hours ?? 0;                                                        |
|                                                                                                       |
| if (h < 1) return `${Math.round(h * 60)}min`;                                                         |
|                                                                                                       |
| return `${h.toFixed(1)}h`;                                                                            |
|                                                                                                       |
| });                                                                                                   |
|                                                                                                       |
| const isFresh = computed(() =>                                                                        |
|                                                                                                       |
| lastFetchAt.value !== null &&                                                                         |
|                                                                                                       |
| !isRecalculating.value &&                                                                             |
|                                                                                                       |
| Date.now() - lastFetchAt.value.getTime() < STALE_MS                                                   |
|                                                                                                       |
| );                                                                                                    |
|                                                                                                       |
| // ── ACTIONS ──────────────────────────────────────────────────────                                  |
|                                                                                                       |
| async function fetchDashboard(force = false): Promise<void> {                                         |
|                                                                                                       |
| if (!force && isFresh.value) return;                                                                  |
|                                                                                                       |
| isLoading.value = true;                                                                               |
|                                                                                                       |
| error.value = null;                                                                                   |
|                                                                                                       |
| try {                                                                                                 |
|                                                                                                       |
| const res = await analyticsApi.getDashboard();                                                        |
|                                                                                                       |
| dashboard.value = res.data;                                                                           |
|                                                                                                       |
| lastFetchAt.value = new Date();                                                                       |
|                                                                                                       |
| } catch (e: any) {                                                                                    |
|                                                                                                       |
| error.value = e.response?.data?.error?.message ?? 'Erro ao carregar dashboard';                       |
|                                                                                                       |
| } finally {                                                                                           |
|                                                                                                       |
| isLoading.value = false;                                                                              |
|                                                                                                       |
| }                                                                                                     |
|                                                                                                       |
| }                                                                                                     |
|                                                                                                       |
| // Chamado pelo WebSocket listener — substitui estado completo                                        |
|                                                                                                       |
| function updateFromWebSocket(newData: DashboardData): void {                                          |
|                                                                                                       |
| dashboard.value = newData;                                                                            |
|                                                                                                       |
| lastFetchAt.value = new Date();                                                                       |
|                                                                                                       |
| isRecalculating.value = false;                                                                        |
|                                                                                                       |
| }                                                                                                     |
|                                                                                                       |
| function setRecalculating(value: boolean): void {                                                     |
|                                                                                                       |
| isRecalculating.value = value;                                                                        |
|                                                                                                       |
| }                                                                                                     |
|                                                                                                       |
| return {                                                                                              |
|                                                                                                       |
| // State (read-only externally — widgets usam computed)                                               |
|                                                                                                       |
| isLoading, isRecalculating, error, selectedPeriod,                                                    |
|                                                                                                       |
| // Computed (granulares por widget)                                                                   |
|                                                                                                       |
| userMetrics, techMetrics, timeSeries, topTechs, weeklyComp,                                           |
|                                                                                                       |
| currentStreak, totalHoursFormatted, isFresh,                                                          |
|                                                                                                       |
| // Actions                                                                                            |
|                                                                                                       |
| fetchDashboard, updateFromWebSocket, setRecalculating,                                                |
|                                                                                                       |
| };                                                                                                    |
|                                                                                                       |
| });                                                                                                   |

3. Composable useDashboard — Lógica da View

O composable encapsula tudo que a View precisa: inicialização, WebSocket, polling de fallback e refresh manual. A View vira um template puro.

+----------------------------------------------------------------------------+
| // src/features/dashboard/composables/useDashboard.ts                      |
|                                                                            |
| import { onMounted, onUnmounted, ref } from 'vue';                         |
|                                                                            |
| import { useAnalyticsStore } from '@/stores/analytics.store';              |
|                                                                            |
| import { useAuthStore } from '@/stores/auth.store';                        |
|                                                                            |
| import { useWebSocket } from '@/composables/useWebSocket';                 |
|                                                                            |
| export function useDashboard() {                                           |
|                                                                            |
| const analyticsStore = useAnalyticsStore();                                |
|                                                                            |
| const authStore = useAuthStore();                                          |
|                                                                            |
| const { connect, disconnect, isConnected } = useWebSocket();               |
|                                                                            |
| let pollingInterval: ReturnType<typeof setInterval> | null = null;         |
|                                                                            |
| async function init(): Promise<void> {                                     |
|                                                                            |
| // 1. Busca dados (usa cache in-memory se frescos)                         |
|                                                                            |
| await analyticsStore.fetchDashboard();                                     |
|                                                                            |
| // 2. Conecta WebSocket                                                    |
|                                                                            |
| connect(authStore.user!.id);                                               |
|                                                                            |
| // 3. Polling de fallback: se WS desconectar, refetch a cada 2 minutos     |
|                                                                            |
| // (garante dados atualizados mesmo sem WebSocket funcionando)             |
|                                                                            |
| pollingInterval = setInterval(() => {                                      |
|                                                                            |
| if (!isConnected.value) {                                                  |
|                                                                            |
| analyticsStore.fetchDashboard(true); // force=true: ignora cache in-memory |
|                                                                            |
| }                                                                          |
|                                                                            |
| }, 2 * 60 * 1000);                                                         |
|                                                                            |
| }                                                                          |
|                                                                            |
| function teardown(): void {                                                |
|                                                                            |
| disconnect();                                                              |
|                                                                            |
| if (pollingInterval) clearInterval(pollingInterval);                       |
|                                                                            |
| }                                                                          |
|                                                                            |
| async function forceRefresh(): Promise<void> {                             |
|                                                                            |
| await analyticsStore.fetchDashboard(true);                                 |
|                                                                            |
| }                                                                          |
|                                                                            |
| // Visibilidade: refetch quando usuário volta para a aba                   |
|                                                                            |
| function handleVisibilityChange(): void {                                  |
|                                                                            |
| if (document.visibilityState === 'visible' && !analyticsStore.isFresh) {   |
|                                                                            |
| analyticsStore.fetchDashboard();                                           |
|                                                                            |
| }                                                                          |
|                                                                            |
| }                                                                          |
|                                                                            |
| onMounted(() => {                                                          |
|                                                                            |
| init();                                                                    |
|                                                                            |
| document.addEventListener('visibilitychange', handleVisibilityChange);     |
|                                                                            |
| });                                                                        |
|                                                                            |
| onUnmounted(() => {                                                        |
|                                                                            |
| teardown();                                                                |
|                                                                            |
| document.removeEventListener('visibilitychange', handleVisibilityChange);  |
|                                                                            |
| });                                                                        |
|                                                                            |
| return { forceRefresh, isConnected };                                      |
|                                                                            |
| }                                                                          |

7. Loading States e Skeleton Loaders

Skeleton loaders são fundamentais para um dashboard profissional. O usuário vê a estrutura da página imediatamente, sem tela em branco ou spinner centralizado que trava a navegação.

7.1 Estados do Dashboard

  ----------------------------------------------------------------------------------------------------
  Estado           Trigger                             Exibição
  ---------------- ----------------------------------- -----------------------------------------------
  Initial Load     onMounted, store vazio              Todos os 5 widgets com SkeletonLoader

  Loaded           fetchDashboard() resolve            Dados reais em todos os widgets

  Recalculating    WS .metrics.recalculating           Overlay semi-transparente + spinner no header

  WS Updated       WS .metrics.updated                 Transição suave: séries/options reativas (ApexCharts)

  Period Switch    Toggle 7d/90d no TimeSeriesWidget   Skeleton só no widget, outros permanecem

  Error            fetchDashboard() rejeita            ErrorCard com botão 'Tentar novamente'

  Disconnected     WS desconecta                       Badge vermelho no header + polling ativo
  ----------------------------------------------------------------------------------------------------
