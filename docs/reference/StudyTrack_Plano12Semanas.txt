-   [ ] GET /api/v1/analytics/dashboard responde em <5ms no segundo request (Redis HIT)

-   [ ] Rate limiting ativo: POST /api/v1/auth/login limitado a 3/min por IP (throttle:login)

-   [ ] WebSocket reconecta automaticamente após queda

-   [ ] Todos os erros retornam o envelope padrão {success:false, error:{code, message}}

-   [ ] Tipos TypeScript sem erros (tsc --noEmit passa)

Portfólio

-   [ ] README com arquitetura explicada e decisões técnicas justificadas

-   [ ] GIF ou vídeo do dashboard em funcionamento

-   [ ] Postman collection com todos os 31 endpoints

-   [ ] DemoDataSeeder gera 6 meses de dados realistas

-   [ ] Deploy acessível publicamente com URL

-   [ ] git log conta a história do desenvolvimento (commits semânticos diários)
